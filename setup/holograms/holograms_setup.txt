# DecentHolograms 2.10.0 Setup Script
# Run from console or RCON (op required).
# Lines starting with # are comments.
#
# Commands:
#   /dh create <name> <x> <y> <z>           - create at world,x,y,z
#   /dh line add <name> "<text>"            - append a line
#   /dh setrange <name> <blocks>            - render distance
#   /dh setupdateinterval <name> <ticks>    - refresh rate (20 = 1s, 100 = 5s)
#   /dh delete <name>
#   /dh list
#
# Placeholder requirements:
#   - PlaceholderAPI must be installed (it is)
#   - Vault expansion:        /papi ecloud download Vault    then  /papi reload
#   - Player expansion:       /papi ecloud download Player
#   - Statistic expansion:    /papi ecloud download Statistic
#   - AuraSkills:             ships its own placeholders (auraskills_*)
#   - VotingPlugin (if installed): votingplugin_*
#
# All holograms place into world "world" by default. Move with:
#   /dh move <name>     (uses your current position)
